I'm going to use Macs built in Apache & I'm not going to install MySQL in this article.

Before you start

You're going to need a few apps on your mac before you can get started.
  1. TextWrangler - This is a great text editor for changing your configuration files on your system & other files. Make sure you download it from their website, the version from the App Store dosen't let you edit system files.
  2. Gas Mask - This lets your easily edit your hosts file so you can point domains such as yoursite.local to your local machine.
  3. Sequel Pro -  This is a handy tool which lets you easily manage MySQL databases.

Turning on Virtual Hosts

Navigate in finder to /private/etc/apache2, you can do this quickly by clicking Go > Go To Folder in the menu bar. Find the file called httpd.conf and than navigate to the line (around 476): [gist id="4141706" file="vhosts-off.sh"] than replace it with [gist id="4141706" file="vhosts-on.sh"]

Turn on PHP

LoadModule php5_module libexec/apache2/libphp5.so

Set up PHPs configuration

php.ini.default > php.ini in /etc

Correct the Virtual Hosts permissions
