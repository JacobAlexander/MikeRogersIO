One of the key improvements is a new backed engine detecting when elements change. The new build should make the extension play more friendly with other extensions which inject HTML into your web page. It also should be more compensating for on load JavaScript page changes. I've also completely rebuilt the options page, it is now a heck of a lot better. You can download it from the Chrome Web Store and you can also contribute to LivePage on GitHub. I've also created a short video demoing the extension. If you have any feedback, feel free to send me a tweet (@MikeRogers0).
